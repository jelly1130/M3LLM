## Datasets

We evaluate M³LLM on **18 public benchmark datasets** spanning **long-term forecasting**, **short-term/zero-shot forecasting**, and **multivariate time-series classification**.

**Long-term forecasting:** ETTm1, ETTm2, ETTh1, ETTh2, Electricity, Weather.  

**Short-term/zero-shot forecasting:** PEMS03, PEMS04, M4-Yearly, M4-Quarterly, M4-Monthly, M4-Weekly, M4-Daily, M4-Hourly, M3-Yearly, M3-Quarterly, M3-Monthly, M3-Others.  

**Classification:** AWR, AF, BL, CR, FM, SRS2.  

The raw datasets are publicly available benchmark datasets. Dataset organization and preprocessing in this project are adapted from the public repository [Time-Series-Library](https://github.com/thuml/Time-Series-Library). We thank the authors of Time-Series-Library for making their code and dataset resources publicly available. The corresponding datasets can be obtained from that repository and its referenced benchmark sources.
